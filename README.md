# 🚀 Java OOP Concepts Project

This repository contains examples of **Object-Oriented Programming (OOP)** concepts implemented in Java.
It is designed for beginners to understand core OOP principles with simple and clear examples.

---

## 📌 About Project

This project demonstrates all major OOP concepts using practical Java programs.
Each concept is explained with real-world examples and clean code.

---

## 🧠 OOP Concepts Covered

### 1️⃣ Abstraction

* Hiding internal implementation and showing only essential features
* Example: ATM Machine

### 2️⃣ Encapsulation

* Wrapping data and methods into a single unit (class)
* Use of getters and setters

### 3️⃣ Inheritance

* One class acquiring properties of another
* Types: Single, Multilevel, Hierarchical

### 4️⃣ Polymorphism

* One name, multiple forms
* Method Overloading & Method Overriding
